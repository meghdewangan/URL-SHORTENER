# Url Shortener

## Using Technology : EJS, BOOTSTRAP, JAVASCRIPT, NODE.JS, MONGODB  

## Process to work :
### Download and install node.js and mongodb
### npm install -- To install all dependencies
### npm run server -- To start server
### The open http://127.0.0.1:5000/
